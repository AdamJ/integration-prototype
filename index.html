Welcome to the Red Hat evaluation environment.

The environment contains everything you need to explore common integration scenarios. Get started with an end-to-end tutorial or dig into any of the available application services on your own to create custom integrations.

Everything you build will reside on the OpenShift cluster associated with this environment.

  • End-to-end integration tutorials
  • One-click access to Red Hat application services, all running on the same OpenShift cluster
