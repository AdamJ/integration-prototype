1.1 Creating an API Connector

  1. Log in to the Red Hat Fuse Online console.
  2. Select Customizations from the left hand menu.
  3. Select the Create API Connector button from the API Client Connector wizard.
  4. When prompted to Upload OpenAPI Specifications, select Use a URL:
    1. Enter the following in the URL field: http://{spring-boot-url}/v2/api-docs
    2. Click Next

Configuring a GitHub OAuth App

  1. Navigate to https://github.com/settings/developers
  2. Click New OAuth App
  3. Enter integration as the name for the app.
  4. Enter the URL you received as prt of your onboarding as the Homepage URL and Authorization callback URL.
